#  Interactive Dashboard Project

## Overview
This project is an interactive dashboard built using Python and Streamlit. It visualizes key business insights using charts and filters.

## Features
- Interactive sidebar filters
- Sales and profit visualization
- Category-wise analysis
- Trend analysis using line charts
- Pie charts for distribution

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly

## Project Structure
app.py  
data.csv  
requirements.txt  

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
